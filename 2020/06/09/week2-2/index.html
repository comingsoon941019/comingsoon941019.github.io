MySQL 외부 접속

  1. aws EC2로 인스턴스한 우분투에 APM 설치

  • Vim 설치

  • APM 설치 https://webnautes.tistory.com/1185

  • MySQL 설치 시에는 root 계정으로 외부접속하는거는 보안상 좋지 않으므로 새로운 계정을 생성
  • 기본적으로 apache2의 디폴트 doc루트는 /var/www/html 이다.
  1. JetBrain DataGrid

  • jet brain 사의 DB를 외부접속하기위한 여러 툴중에 하나 이다.

  • 한달 동안 무료로 사용가능하며 대학생일 경우 대학교 이메일을 인증하면 1년간 Professional linecese를 이용 할 수 있다.

  • 아래 DB source 설정에서 mysql user와 비밀번호 EC2 인스턴스의 아이피를 적어 주어 연결한다.
