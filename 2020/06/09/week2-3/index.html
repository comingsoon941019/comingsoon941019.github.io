서버 2주차

aws EC2 도메인 적용

https 보안 인증

  1. 로컬 윈도우 컴퓨터에 비트나미로 서버 구축, → 폰 와이파이 풀고 Phpinfo 외부접속 캡쳐
    이후 과제는 모두 aws 컴퓨터에 시행, aws ec2 임대, nginx, php, mysql 패키지 관리자로 설치
  1. 도메인 적용
  2. https 적용 (let’s encrypt)
  3. 웹템플릿 적용해서 나만의 웹사이트 만들어오기 → HTML, CSS, JS 활용

aws EC2 도메인 적용

  1. 무료 도메인 Freenom : https://www.freenom.com/en/index.html?lang=en

  • 회원가입후 원하는 도메인을 설정 무료로 사용하기 위해서 free표시를 꼭 확인하자

  • aws 서비스에서 Route53 서비스 이용

  • 호스트 영역을 생성후 다음과 같이 name serve를 얻을 수 있다.

  • 이제 다시 freenom 사이트에서 발급받은 네임 서버를 등록해 준다.

  1. https 인증

  • Certificate Manager 서비스를 사용하여 인증서를 받는다

  • 인증서를 문제 없이 받았다면 EC2 서비스의 로드 밸런서에서 도메인과 https 포트를 등록하며 방금 받은 인증서를 등록한다.

  • 모든 설정이 끝난후에 https:// 로 접속하면 확인 가능하다. 다만 항상 EC2인스턴스 설정에 포트를 열어주는 것을 잊지말자…
