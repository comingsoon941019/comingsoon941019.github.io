Apache compile install

Vim install, setting

  • 우분투를 사용할 때 vi 편집기를 사용할 때 vim설치를 해주지 않고 사용하면 매우매우 불편하다. 먼저 설치를 하고 시작하자.

  • apt 업데이트 이후 설치

‘’’bash
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install vim
‘’’

  • vim 설치가 끝이 아니라 설정까지 해줘야 vim사용이 가능하다.

  • vim설정 파일 진입

‘’’bash
$ vi ~/.vimrc
‘’’

‘’’bash
set number “ line 표시
set ai “ auto indent
set si “ smart indent
set cindent “ c style indent
set shiftwidth=4 “ 자동 공백 채움 시 4칸
set tabstop=4 “ tab을 4칸 공백으로
set ignorecase “ 검색 시 대소문자 무시
set hlsearch “ 검색 시 하이라이트
set nocompatible “ 방향키로 이동 가능
set fencs=ucs-bom,utf-8,euc-kr “ 한글 파일은 euc-kr, 유니코드는 유니코드
set bs=indent,eol,start “ backspace 사용가능
set ruler “ 상태 표시줄에 커서 위치 표시
set title “ 제목 표시
set showmatch “ 다른 코딩 프로그램처럼 매칭되는 괄호 보여줌
set wmnu “ tab 을 눌렀을 때 자동완성 가능한 목록
syntax on “ 문법 하이라이트 on
filetype indent on “ 파일 종류에 따른 구문 강조
set mouse=a “ 커서 이동을 마우스로 가능하도록
‘’’

가상환경에 우분투 설치

Virtual Box

  • mac을 사용하고 있지만 ubuntu 공부를 위해 가상환경에서 apm 설치를 하고자 한다.

  • 가상환경은 무료로 사용가능한 Virtual Box를 사용한다.

Ubuntu 20.04 LTS

  • 우분투 (ubuntu)는 데비안 GNU/리눅스 기반으로 개발된 배포판

  • LTS(장기 지원판)이며 2년에 한번씩 배포됨

  • 해당 사이트에서 이미지 파일을 받아 가상머신에 설치 할 예정 다운받는 속도는 매우 느리다.

virtual box 에 ubuntu 설치

  • virtual box에 새로만들기 클릭후 가상머신의 이름과 버젼, 종류를 선택

  • 이후 메모리 크기를 설정 하는데 공홈에서 4GB를 권장하지만 여유가 된다면 이상을 추천한다.

  • 하드 디스크 지정 후 VDI 선택

  • 하드 디스크를 동적 할당 또는 고정크기로 정하는데 호스트 컴퓨터 하드디스크에 여유가 없음 동적 할당을 추천

  • 다음으로 하드의 크기를 정할때가 문제 생각 이상으로 apm 설치가 용량이 많이 들어 최소 40기가 정도로 설정하는게 번거롭게 재설치를 하지 않아도 될듯하다.

  • 다음으로 아까 받았던 ubuntu img를 올려 설치를 시작한다.

  • 이제 설치를 하다 보면 우분투 해상도 문제로 버튼을 클릭하지 못하는 문제가 발생하는데 이때 우분투를 종료후 virtualbox의 디스플레이 설정에서 그래픽컨트롤러를 VBoxSVGA로 변경
